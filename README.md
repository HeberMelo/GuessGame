# Guess Game
Aplicación de escritorio desarrollada en Python que consiste en un juego interactivo donde el usuario debe adivinar un número secreto entre 1 y 100, eligiendo previamente el nivel de dificultad.
Este proyecto fue creado con fines educativos, para practicar lógica de programación, manejo de eventos y desarrollo de interfaces gráficas con Python.

# Características
Cuenta con un selector de dificultad.
  • **Fácil** (10 intentos)
  
  • **Medio** (5 intentos)
  
  • **Difícil** (3 intentos)
  
• El usuario tiene que adivinar el número elegido entre 1 y 100
  
• Contador de intentos restantes
  
• Mensajes de ayuda (el número es mayor o menor)
  
• Botón para reiniciar el juego
  
• Interfaz gráfica hecha con customtkinter

# Tecnologías utilizadas.
• Python 3
• CustomTkinter
